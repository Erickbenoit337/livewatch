⚙️ Paramètres

Personnalisez votre expérience {{ app_name }}. Tous les paramètres sont sauvegardés localement sur votre appareil.

📱 Économie de données mobiles

💡 À quoi ça sert ? Sur une connexion mobile (4G/5G), chaque vidéo consomme des mégaoctets. Ces options vous permettent de regarder {{ app_name }} en utilisant moins de données.

🎬 Qualité vidéo par défaut
Auto : s'adapte à votre connexion. SD (480p) : environ 3× moins de données que HD. HD (720p) : qualité standard. Full HD : meilleure qualité, consomme le plus.
▶️ Lecture automatique
Si activée, les vidéos se lancent dès l'ouverture d'une chaîne. Désactivez pour économiser des données : vous choisissez vous-même quand démarrer.
⏩ Préchargement des thumbnails
Les miniatures (vignettes) des chaînes sont chargées à l'avance pour un affichage plus rapide. Désactivez sur mobile pour économiser des données — les images se chargent à la demande uniquement.
🔋 Mode économie de données
Active automatiquement SD + désactive le préchargement + réduit les animations. Recommandé sur réseau mobile ou connexion lente. Économise jusqu'à 70% des données.
✅ Mode économie actif : SD forcé, thumbnails à la demande, animations réduites.
🎬 Lecteur vidéo
🔊 Volume par défaut
Volume appliqué automatiquement à l'ouverture d'une chaîne (0 = muet, 100 = plein volume).
80%
📝 Sous-titres automatiques
Active les sous-titres si disponibles sur la chaîne. Utile pour les chaînes en langue étrangère.
🖥️ Plein écran automatique
Passe automatiquement en plein écran dès qu'une chaîne démarre. Pratique sur tablette ou TV connectée.
🎨 Interface & Affichage
🌙 Thème
Le mode sombre réduit la fatigue oculaire et consomme moins de batterie sur les écrans OLED.
🌐 Langue préférée des chaînes
Filtre les chaînes recommandées selon la langue choisie. N'empêche pas de regarder les autres langues.
✨ Animations de l'interface
Désactivez pour une interface plus rapide, notamment sur les appareils moins puissants ou les connexions lentes.
🔔 Notifications
📅 Rappels programmes TV
Recevez une notification 5 minutes avant le début d'un programme que vous avez mis en rappel depuis la page Événements.
🔒 Confidentialité

💡 {{ app_name }} ne collecte aucune donnée personnelle identifiable. Aucun compte requis. Vos préférences sont stockées uniquement sur votre appareil.

🕐 Historique de visionnage local
Garde en mémoire les dernières chaînes regardées pour un accès rapide. Stocké uniquement sur votre appareil, jamais envoyé.
🗑️ Effacer toutes les données locales
Remet tous les paramètres à zéro et efface l'historique, les favoris locaux et les préférences. Action irréversible.
